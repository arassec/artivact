Artivact User Manual ​

About ​

Work in progress

The user manual is currently in creation and will be updated with future releases.

The Artivact user manual documents how to use the application to create and manage your virtual collections.

The desktop and web server variant of the application differ slightly in their feature sets. If a function only applies to one of the installation variants, this will be denoted in the documentation by one of the following badges:

  • server - This functionality is only available, if Artivact is run in web server mode.
  • desktop - This functionality is only available, if Artivact is started on your local computer.

The Main Screen ​

After first start, the main screen is shown to the user. At the top is the navigation and settings bar.

The following functions are available from left to right:

ButtonDescription
Adds a new menu to the top navigation. See Menu Management for details.
Selects the locale for internationalization (I18N). See TODO for details.
Opens the items menu. From there you can create new items or import existing ones. See TODO for details.
Opens the exhibitions configuration. From there you can create and manage exhibitions of your collection. See TODO for details.
Opens the system settings menu. See TODO for details.
server Opens the account settings mennu. See TODO for details.
Opens this documentation from within the application.
server Logs the user out of the application.
server Opens the login page.

You can use the plus button to add a new menu entry and define a page as described in the next chapter.
