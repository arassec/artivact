Artivact Turntable ​

Automatic Object Rotation ​

In order to speedup item creation, and since the camera can be automatically controlled, the object can be automatically rotated by Artivact as well.

Based on other do-it-yourself 3D scanner solutions in the web, Artivact provides an automatic 3D turntable.

The parts have to be 3D printed and assembled by yourself. It can afterward be used to rotate the object automatically between image capturing.

Required Parts ​

The following parts are required to assemble the turntable.

PartDescriptionDownload/Example
Turntable PartsThe parts of the turntable that have to be 3D printed and assembled.Github
Stepper Motor 28BYJ-48 + Driver ULN2003The stepper motor to rotate the turntable.Amazon
Rotary Bearing Swivel Plate 360° Degree4 Inch Square Swivel to support heavy objects.Amazon
Arduino Nano EveryUsed to control the stepper motor from PC via USB.Arduino Store

Arduino Sketch

The Arduino Nano needs to have the following sketch installed: sketch_artivact-tt-v1.ino. Please refer to the Arduino Documentation for how to do this.
