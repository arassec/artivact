# Internationalization (I18N)

## Configuration

Supported locales can be configured in the [Appearance Configuration](/user-manual/settings/appearance).

A locale, to edit items and pages for, can be selected by the globe symbol:
![locale-selection-button](../introduction/assets/about/locale-selection-button.png)

When selected, all text inputs are then edited for the selected locale.

The default locale's value, which is used as fallback if the user's preferences don't match any supported locale, will
be shown to the right of the input field.

## Automatic Translation with AI

If AI integration is enabled and configured, Artivact can automatically translate texts into the configured locales.
This includes item titles, descriptions, and page content.

The translation behavior can be customized via the **Translation Prompt** and **General Context** in the
[AI Configuration](../settings/ai).

::: tip AI Translation
Automatic AI translation can significantly speed up the process of maintaining multilingual content. The translations
can always be manually reviewed and adjusted afterwards.
:::
